# 筑梦管理_多媒体端
英文名CreatingDreamsManagement(CDM)  
原名 校园多媒体统一管理系统  
这里是客户端，使用electron开发。服务端请[点击这里](https://github.com/heyManNice/CreatingDreamsManagement)  
# 启动
./main.js中第8行ip改为你部署的服务器ip  
安装nodejs和npm之后  
在./执行npm install自动安装依赖  
执行npm test运行  
或者双击./run.exe
